# e-plantShopping

## Paradise Nursery Shopping Application

e-plantShopping is a React and Redux based shopping cart application for an online plant nursery called Paradise Nursery.

The application allows users to:

- Browse different categories of houseplants
- View plant images, names, and prices
- Add plants to the shopping cart
- Increase or decrease plant quantities
- Remove items from the cart
- View the total number of plants and total cart amount

## Technologies Used

- React
- Redux Toolkit
- React Router
- CSS

## Features

- Landing page with background image
- Product listing page with categorized plants
- Dynamic shopping cart
- Responsive navigation bar
- Checkout and Continue Shopping buttons
